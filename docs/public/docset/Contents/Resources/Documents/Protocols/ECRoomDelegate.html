– room:didAddedStreamId: required method

Event fired once a new stream has been added to the room.

- (void)room:(ECRoom *)room didAddedStreamId:(NSString *)streamId

Parameters

room

Instance of the room where event happen.
sreamId

The stream id of the added stream.

Discussion

It is up to you to subscribe that stream or not. It is worth to notice that your published stream will not be notified by this method, use ECRoomDelegate:didPublishStreamId: instead.
