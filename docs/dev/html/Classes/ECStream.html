Overview

@interface ECStream

Represents a wrapper around an audio/video RTC stream that can be used to access local media and publish it in a ECRoom, or receive video from.

Initializers

– initWithLocalStreamWithMediaConstraints:

Creates an instance of ECStream capturing audio/video data from host device.

- (instancetype)initWithLocalStreamWithMediaConstraints:(RTCMediaConstraints *)mediaConstraints

Parameters

mediaConstraints

RTCMediaConstraints that apply to this stream.

Return Value

instancetype

Discussion

Inmediatly attempts to acceess local audio/video. You can pass nil to mediaConstraints and default media constraints will be used.
